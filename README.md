Excel 365 new functions examples / explorations - includes interfaces of excel to python / java / java script / c++ / c / rust / standalone python scrips, etc. - updated till 21-10-2025 (dd-mm-yyyy). Excel to other languages interfaces (C / C++ / Java / Rust, etc.) are through python libraries as bridge.

Special addition: Python rdrand (supporting intel intrinsic HRNG crytography supported random number generator functions - rdrand and rdseed). They also work on AMD. Original package is at https://github.com/stillson/rdrand which did not supporting building on msvc ecosystem (only supported gnu gcc / clang environment) and development stopped long ago. This improvised fork has msvc supported source and build. In case, only interested in using binaries, they have also been included inside the folder rdrand-1.5.0 (along with other source files / folders). Binaries and building is done on latest python 3.14 version just released.
